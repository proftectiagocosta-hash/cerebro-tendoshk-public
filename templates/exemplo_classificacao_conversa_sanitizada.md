# Modelo de Classificação de Conversa Sanitizada / Sanitized Conversation Classification Template

[Português](#português) | [English](#english)

---

## Português

Este modelo demonstra como uma conversa fictícia pode ser classificada sem expor conteúdo real, dados pessoais ou material operacional privado.

### Identificação

- **Título fictício:** Nome seguro da conversa
- **Data aproximada:** AAAA-MM-DD
- **Área:** projeto, estudo, criação, decisão, referência ou outro
- **Sensibilidade:** baixa, média ou alta

### Valor estrutural

Descreva se a conversa contém algo que mereça preservação:

- decisão importante;
- ideia reutilizável;
- próximo passo;
- mudança de direção;
- problema recorrente;
- aprendizado;
- referência futura.

### Classificação recomendada

- [ ] Manter como referência simples
- [ ] Transformar em entrada de memória
- [ ] Transformar em checkpoint
- [ ] Transformar em documentação
- [ ] Reaproveitar em projeto público
- [ ] Arquivar
- [ ] Descartar

### Riscos de exposição

- [ ] Contém dados pessoais
- [ ] Contém conversas privadas
- [ ] Contém caminhos internos
- [ ] Contém decisões sensíveis
- [ ] Contém material operacional privado
- [ ] Pode ser sanitizado com segurança

### Saída sanitizada

Registre apenas a versão pública, segura e reutilizável do conteúdo.

### Próxima ação

Defina uma ação simples e rastreável:

- registrar;
- revisar;
- converter;
- resumir;
- transformar em documentação;
- manter privado;
- descartar.

## English

This template demonstrates how a fictional conversation can be classified without exposing real content, personal data, or private operational material.

### Identification

- **Fictional title:** Safe conversation name
- **Approximate date:** YYYY-MM-DD
- **Area:** project, study, creation, decision, reference, or other
- **Sensitivity:** low, medium, or high

### Structural value

Describe whether the conversation contains something worth preserving:

- important decision;
- reusable idea;
- next step;
- direction change;
- recurring problem;
- learning;
- future reference.

### Recommended classification

- [ ] Keep as simple reference
- [ ] Convert into memory entry
- [ ] Convert into checkpoint
- [ ] Convert into documentation
- [ ] Reuse in public project
- [ ] Archive
- [ ] Discard

### Exposure risks

- [ ] Contains personal data
- [ ] Contains private conversations
- [ ] Contains internal paths
- [ ] Contains sensitive decisions
- [ ] Contains private operational material
- [ ] Can be safely sanitized

### Sanitized output

Record only the public, safe, and reusable version of the content.

### Next action

Define a simple and traceable action:

- register;
- review;
- convert;
- summarize;
- turn into documentation;
- keep private;
- discard.
