# Segurança e Privacidade / Security and Privacy

[Português](#português) | [English](#english)

---

## Português

Este repositório é público e deve conter apenas documentação sanitizada.

### Não publique aqui

Nunca envie:

- tokens;
- senhas;
- chaves de API;
- arquivos `.env`;
- dados pessoais;
- conversas privadas;
- histórico emocional ou familiar;
- checkpoints reais;
- prompts privados;
- arquivos operacionais privados;
- caminhos internos de Drive, máquina ou serviços;
- credenciais ou configurações secretas.

### Regra de ouro

Se um arquivo permitir reconstruir uma estrutura interna sensível, ele não pertence a este repositório público.

### Material permitido

Este repositório pode conter:

- exemplos fictícios;
- modelos genéricos;
- fluxos abstratos;
- documentação conceitual;
- diagramas sem dados privados reais;
- itens de roadmap público.

### Reporte

Se encontrar material sensível neste repositório, abra uma issue ou entre em contato pelo canal público disponível do mantenedor.

## English

This repository is public and must contain only sanitized documentation.

### Do not publish here

Never commit:

- tokens;
- passwords;
- API keys;
- `.env` files;
- personal data;
- private conversations;
- emotional or family history;
- real checkpoints;
- private prompts;
- private operational files;
- internal Drive, machine, or service paths;
- credentials or secret configuration.

### Golden rule

If a file allows someone to reconstruct sensitive internal structure, it does not belong in this public repository.

### Allowed material

This repository may contain:

- fictional examples;
- generic templates;
- abstract flows;
- conceptual documentation;
- diagrams without real private data;
- public roadmap items.

### Reporting

If you find sensitive material in this repository, please open an issue or contact the maintainer through the available public channel.
