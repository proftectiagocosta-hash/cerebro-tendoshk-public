# Princípios Públicos / Public Principles

[Português](#português) | [English](#english)

---

## Português

### 1. Continuidade sem exposição

O projeto valoriza continuidade, mas não ao custo da privacidade.

### 2. A camada pública não é o núcleo privado

Este repositório explica conceitos. Ele não contém o sistema operacional completo.

### 3. Apenas exemplos fictícios

Modelos e exemplos devem ser sanitizados e não identificáveis.

### 4. Curadoria acima de acumulação

O objetivo não é armazenar tudo. O objetivo é preservar o que permanece útil, estruturado e seguro.

### 5. Arquitetura centrada no humano

Memória assistida por IA deve apoiar o julgamento humano, não substituí-lo.

### 6. Rastreabilidade importa

Decisões e transições importantes devem ser registradas de forma revisável posteriormente.

### 7. Segurança antes de visibilidade

Um repositório público deve esclarecer a visão sem expor material operacional sensível.

---

## English

### 1. Continuity without exposure

The project values continuity, but not at the cost of privacy.

### 2. Public layer is not the private core

This repository explains concepts. It does not contain the complete operational system.

### 3. Fictional examples only

Templates and examples must be sanitized and non-identifying.

### 4. Curation over accumulation

The goal is not to store everything. The goal is to preserve what remains useful, structured, and safe.

### 5. Human-centered architecture

AI-assisted memory should support human judgment, not replace it.

### 6. Traceability matters

Important decisions and transitions should be recorded in a way that can be reviewed later.

### 7. Safety before visibility

A public repository should clarify the vision without exposing sensitive operating material.
