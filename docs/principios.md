# Public Principles

## 1. Continuity without exposure

The project values continuity, but not at the cost of privacy.

## 2. Public layer is not the private core

This repository explains concepts. It does not contain the complete operational system.

## 3. Fictional examples only

Templates and examples must be sanitized and non-identifying.

## 4. Curation over accumulation

The goal is not to store everything. The goal is to preserve what remains useful, structured, and safe.

## 5. Human-centered architecture

AI-assisted memory should support human judgment, not replace it.

## 6. Traceability matters

Important decisions and transitions should be recorded in a way that can be reviewed later.

## 7. Safety before visibility

A public repository should clarify the vision without exposing sensitive operating material.
