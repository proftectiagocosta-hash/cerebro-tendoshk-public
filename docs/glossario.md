# Glossário / Glossary

[Português](#português) | [English](#english)

---

## Português

### Cérebro externo

Sistema de apoio para preservar contexto, decisões, projetos e memórias úteis.

### Núcleo privado

Material operacional sensível que não deve ser público.

### Camada pública

Documentação sanitizada que explica o projeto sem expor estrutura privada.

### Checkpoint

Registro de estado usado para retomar um projeto ou linha de trabalho posteriormente.

### Curadoria

Processo de avaliar, filtrar, organizar e transformar informação bruta em memória útil.

### Memória persistente

Informação organizada para reutilização futura.

### Modelo sanitizado

Modelo ou exemplo que remove dados pessoais, sensíveis, operacionais ou identificáveis.

### Continuidade

Capacidade de retomar um projeto, raciocínio ou fluxo de trabalho sem perder o contexto anterior.

### Rastreabilidade

Capacidade de revisar quando, por que e como uma decisão relevante ou transição de estado aconteceu.

## English

### External brain

A support system for preserving context, decisions, projects, and useful memory.

### Private core

Sensitive operational material that should not be public.

### Public layer

Sanitized documentation that explains the project without exposing private structure.

### Checkpoint

A state record used to resume a project or line of work later.

### Curation

The process of evaluating, filtering, organizing, and transforming raw information into useful memory.

### Persistent memory

Information organized for future reuse.

### Sanitized template

A model or example that removes personal, sensitive, operational, or identifying data.

### Continuity

The ability to resume a project, thought process, or workflow without losing its previous context.

### Traceability

The ability to review when, why, and how a relevant decision or state transition happened.
