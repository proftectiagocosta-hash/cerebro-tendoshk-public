# Glossary

## External brain

A support system for preserving context, decisions, projects, and useful memory.

## Private core

Sensitive operational material that should not be public.

## Public layer

Sanitized documentation that explains the project without exposing private structure.

## Checkpoint

A state record used to resume a project or line of work later.

## Curation

The process of evaluating, filtering, organizing, and transforming raw information into useful memory.

## Persistent memory

Information organized for future reuse.

## Sanitized template

A model or example that removes personal, sensitive, operational, or identifying data.

## Continuity

The ability to resume a project, thought process, or workflow without losing its previous context.

## Traceability

The ability to review when, why, and how a relevant decision or state transition happened.
