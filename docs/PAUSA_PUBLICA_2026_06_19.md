# Pausa pública do Cérebro Tendoshk / Public pause of Cérebro Tendoshk

Data: 2026-06-19  
Estado público: vitrine preservada  
Versão pública preservada: 0.2.2  
Escopo: documentação pública sanitizada, conceitual e não operacional

---

## Português

### 1. Decisão pública

A camada pública do Cérebro Tendoshk passa a ser tratada como vitrine preservada e documentação conceitual estável.

Esta pausa pública acompanha a pausa operacional do núcleo privado, sem expor detalhes internos, histórico sensível, registros reais, automações, checkpoints privados ou estruturas reservadas.

### 2. O que permanece válido

A versão pública `0.2.2` continua válida como marco de documentação conceitual sanitizada.

Permanecem válidos:

- a visão pública do Cérebro Tendoshk;
- a separação entre camada pública e núcleo reservado;
- os princípios de continuidade, curadoria e governança de estado;
- os exemplos sanitizados;
- os casos de uso públicos;
- a vitrine HTML como material de apresentação.

### 3. Limites durante a pausa

Enquanto esta pausa pública estiver vigente:

- novas versões públicas não devem ser publicadas por atualização cosmética;
- avanços privados não devem ser copiados automaticamente para esta camada pública;
- nenhum material sensível deve ser exposto;
- este repositório deve funcionar como vitrine, referência e documentação pública preservada;
- qualquer retomada pública deve exigir decisão explícita e avanço conceitual real.

### 4. Síntese

A camada pública não está abandonada.

Ela está preservada como vitrine conceitual segura do ciclo já publicado.

## English

### 1. Public decision

The public layer of Cérebro Tendoshk is now treated as a preserved showcase and stable conceptual documentation surface.

This public pause follows the operational pause of the private core without exposing internal details, sensitive history, real records, private checkpoints, automations, or reserved structures.

### 2. What remains valid

Public version `0.2.2` remains valid as a milestone of sanitized conceptual documentation.

The following remain valid:

- the public vision of Cérebro Tendoshk;
- the separation between the public layer and the reserved core;
- the principles of continuity, curation, and state governance;
- sanitized examples;
- public use cases;
- the HTML showcase as presentation material.

### 3. Limits during the pause

While this public pause is active:

- new public versions should not be published for cosmetic updates;
- private progress should not be copied automatically into this public layer;
- no sensitive material should be exposed;
- this repository should function as a preserved showcase, reference, and public documentation surface;
- any public resumption should require an explicit decision and real conceptual progress.

### 4. Summary

The public layer is not abandoned.

It is preserved as a safe conceptual showcase of the already published cycle.
