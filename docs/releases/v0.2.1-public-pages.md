# v0.2.1 — Public Pages Showcase

[Português](#português) | [English](#english)

---

## Português

Esta versão registra a publicação e validação da vitrine pública do Cérebro Tendoshk no GitHub Pages, incluindo a navegação interna HTML da vitrine e a publicação dos exemplos sanitizados de linhas vivas.

### Foco da versão

A versão `v0.2.1-public-pages` consolida a passagem da vitrine pública de documentação versionada para publicação pública acessível.

Ela mantém a versão conceitual `0.2.0` como base de governança pública de continuidade e acrescenta o marco público de publicação da vitrine, navegação interna HTML, página pública de governança de estado e página pública de linhas vivas.

### O que mudou

- A vitrine pública foi publicada via GitHub Pages.
- A URL pública foi confirmada: `https://proftectiagocosta-hash.github.io/cerebro-tendoshk-public/`.
- O site público foi alinhado à linguagem da versão `0.2.0`.
- Os links do site foram ajustados para funcionar corretamente quando apenas a pasta `site/` é publicada.
- A vitrine passou a usar páginas HTML internas para navegação principal.
- Foi adicionada página pública sanitizada sobre governança de estado.
- Foram adicionados exemplos sanitizados de linhas vivas de projeto.
- A página interna `site/linhas-vivas.html` foi publicada e validada.
- A regra `linhas_vivas_publicas=sim` foi validada no workflow Pages.
- A navegação interna HTML foi validada via GitHub Pages.
- O README público passou a destacar o site publicado e a documentação de linhas vivas.
- O roadmap e o changelog foram atualizados para refletir a publicação real da vitrine, sua navegação interna e as linhas vivas.
- O workflow de publicação de release passou a apontar por padrão para esta release pública.

### Segurança

A publicação continua sendo uma vitrine sanitizada. Ela não contém núcleo operacional privado, dados pessoais, registros reais de operação, arquivos reservados, checkpoints privados ou mecanismos internos completos.

### Resultado

```txt
Publicação pública = concluída
GitHub Pages = validado
Vitrine v0.2.1 = publicada
Navegação interna HTML = validada
Governança de estado pública = publicada
Linhas vivas públicas = publicadas e validadas
Release pública = v0.2.1-public-pages
```

## English

This release records the publication and validation of the public Cérebro Tendoshk showcase on GitHub Pages, including the showcase's internal HTML navigation and the publication of sanitized living-lines examples.

### Release focus

Version `v0.2.1-public-pages` consolidates the transition from versioned public documentation to an accessible public showcase.

It keeps conceptual version `0.2.0` as the foundation for public continuity governance and adds the public milestone of showcase publication, internal HTML navigation, a public state governance page, and a public living-lines page.

### What changed

- The public showcase was published through GitHub Pages.
- The public URL was confirmed: `https://proftectiagocosta-hash.github.io/cerebro-tendoshk-public/`.
- The public site was aligned with the `0.2.0` language.
- Site links were adjusted to work correctly when only the `site/` folder is published.
- The showcase now uses internal HTML pages for main navigation.
- A sanitized public page about state governance was added.
- Sanitized examples of living project lines were added.
- Internal page `site/linhas-vivas.html` was published and validated.
- Rule `linhas_vivas_publicas=sim` was validated in the Pages workflow.
- Internal HTML navigation was validated through GitHub Pages.
- The public README now highlights the published site and the living-lines documentation.
- The roadmap and changelog were updated to reflect the actual public showcase publication, internal navigation, and living lines.
- The release publication workflow now defaults to this public release.

### Security

The publication remains a sanitized public showcase. It does not contain the private operational core, personal data, real operational records, reserved files, private checkpoints, or complete internal mechanisms.
