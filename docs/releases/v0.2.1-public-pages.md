## Português

Esta versão registra a publicação e validação da vitrine pública do Cérebro Tendoshk no GitHub Pages.

### Foco da versão

A versão `v0.2.1-public-pages` consolida a passagem da vitrine pública de documentação versionada para publicação pública acessível.

Ela mantém a versão conceitual `0.2.0` como base de governança pública de continuidade e acrescenta o marco operacional público de publicação da vitrine.

### O que mudou

- A vitrine pública foi publicada via GitHub Pages.
- A URL pública foi confirmada: `https://proftectiagocosta-hash.github.io/cerebro-tendoshk-public/`.
- O site público foi alinhado à linguagem da versão `0.2.0`.
- Os links do site foram ajustados para funcionar corretamente quando apenas a pasta `site/` é publicada.
- O README público passou a destacar o site publicado.
- O roadmap e o changelog foram atualizados para refletir a publicação real da vitrine.
- O workflow de publicação de release passou a apontar por padrão para esta release pública.

### Segurança

A publicação continua sendo uma vitrine sanitizada. Ela não contém núcleo operacional privado, dados pessoais, registros reais de operação, arquivos reservados ou mecanismos internos completos.

## English

This release records the publication and validation of the public Cérebro Tendoshk showcase on GitHub Pages.

It keeps conceptual version `0.2.0` as the foundation for public continuity governance and adds the public publication milestone.

### What changed

- The public showcase was published through GitHub Pages.
- The public URL was confirmed: `https://proftectiagocosta-hash.github.io/cerebro-tendoshk-public/`.
- The public site was aligned with the `0.2.0` language.
- Site links were adjusted to work correctly when only the `site/` folder is published.
- The public README now highlights the published site.
- The roadmap and changelog were updated to reflect the actual public showcase publication.
- The release publication workflow now defaults to this public release.

### Security

The publication remains a sanitized public showcase. It does not contain the private operational core, personal data, real operational records, reserved files, or complete internal mechanisms.
