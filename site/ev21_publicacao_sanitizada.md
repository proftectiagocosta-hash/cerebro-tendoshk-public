# EV-21 — Publicação Pública Sanitizada

## Natureza

Este documento registra a intenção e os limites da publicação pública da EV-21 no repositório:

```txt
proftectiagocosta-hash/cerebro-tendoshk-public
```

## Escopo

```txt
EV-21 — GitHub Pages / publicação controlada da vitrine pública sanitizada
```

A publicação usa uma camada pública própria, separada do núcleo operacional privado.

## Motivo

O repositório operacional privado permanece protegido. A publicação pública deve apresentar a visão do Cérebro Tendoshk sem expor arquivos internos, dados privados, logs reais, relatórios privados ou estrutura sensível.

## Permitido

```txt
visão pública
arquitetura conceitual
princípios
roadmap público
modelos fictícios
links para documentação pública
aviso de sanitização
```

## Proibido

```txt
RAWs
chats privados
prompts privados
checkpoints reais
operations_log real
reports reais
interface_data real do núcleo privado
arquivos de estado privados
histórico operacional privado
segredos ou credenciais
caminhos internos
```

## Critério de validação

A EV-21 só deve ser concluída quando:

```txt
1. site/index.html existir;
2. workflow de GitHub Pages existir;
3. workflow publicar apenas site/;
4. validação de segurança passar;
5. deploy gerar URL pública;
6. URL abrir corretamente;
7. registro final for criado no repositório privado.
```

## Regra

```txt
A publicação pública é vitrine sanitizada, não espelho do núcleo operacional privado.
```
